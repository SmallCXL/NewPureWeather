# NewPureWeather

识雨晴天气是一款功能简洁、界面鲜活的看天气应用。软件页面的设计遵循了谷歌的Material Design风格，没有植入广告，专注为您提供24小时天气实况报道，间隔三小时和七天之内的天气预测，以及智能定位功能。如果你厌倦了目前市场上广告繁多的商业软件，识雨晴天气将是您最好的选择。
